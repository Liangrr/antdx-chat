# Ant Design X 风格 AI 聊天界面设计说明

## 🎨 界面布局

本项目采用了类似 Ant Design X 官网的三栏式布局设计：

```
┌─────────────────────────────────────────────────────────────┐
│  [左侧边栏]  │        [主内容区]        │  [右侧指南]  │
│              │                          │              │
│  Ant Design X│    你好，我是 Ant Design X│   设计指南   │
│              │                          │              │
│  + 新对话    │    基于蚂蚁设计，AGI产品  │   □ 意图     │
│              │    新解决方案...          │   □ 角色     │
│  今天        │                          │   □ 对话     │
│  □ 对话1     │    热门话题              │   □ 界面     │
│  □ 对话2     │    1 Ant Design X...     │              │
│              │    2 新的 AGI 混合界面    │              │
│  昨天        │    3 Ant Design X...     │              │
│  □ 对话3     │    4 快速发现 AI...      │              │
│              │    5 如何快速安装...      │              │
│              │                          │              │
│              │  ┌─────────────────────┐ │              │
│              │  │ 📎 提问或输入 / 使用 │ │              │
│              │  │    指令     🎤  ➤   │ │              │
│              │  └─────────────────────┘ │              │
│              │  升级 组件 RICH指南 安全  │              │
└─────────────────────────────────────────────────────────────┘
```

## 📐 设计特点

### 1. 左侧边栏（Sidebar）
- **宽度**: 260px
- **背景色**: #fafafa
- **功能**:
  - Logo 和品牌名称
  - "新对话" 按钮（主色调）
  - 对话历史列表，按时间分组（今天、昨天、更早）
  - 每个对话项支持悬停显示更多操作（重命名、删除）
  - 当前选中对话高亮显示

### 2. 主内容区（Main Content）
#### 欢迎页（无对话时）
- **居中布局**
- **元素**:
  - AI 头像（渐变色圆形）
  - 欢迎标题："你好，我是 Ant Design X"
  - 副标题说明
  - 热门话题卡片网格（带序号标签）
  - 卡片悬停效果（上浮 + 阴影）

#### 对话页（有对话时）
- **最大宽度**: 900px，居中显示
- **消息样式**:
  - 简洁设计，无边框背景
  - 用户/AI 头像区分
  - 消息操作按钮（复制、重试、删除）悬停显示

### 3. 右侧指南面板（Guide Panel）
- **宽度**: 280px
- **背景色**: #fafafa
- **内容**:
  - "设计指南" 标题
  - 渐变色卡片容器
  - 四个指南项（意图、角色、对话、界面）
  - 每项包含图标、标题、描述
  - 悬停效果（左移 + 阴影）

### 4. 输入区（Chat Input）
#### 工具栏
- 位于输入框上方
- 包含：升级、组件、RICH 指南、安全介绍等按钮
- 小尺寸文本按钮

#### 输入框
- **背景色**: #fafafa（未聚焦），#fff（聚焦）
- **边框**: 圆角 12px
- **布局**: 横向排列
  - 📎 附件按钮（左侧）
  - 文本输入区（中间，自适应高度）
  - 🎤 语音按钮（右侧）
  - ➤ 发送按钮（圆形，主色调）
- **交互**:
  - 聚焦时边框变蓝色，添加阴影
  - Enter 发送，Shift+Enter 换行

## 🎨 色彩方案

### 主色调
- **品牌色**: #1890ff（蓝色）
- **渐变色**: 
  - 头像/标签：#667eea → #764ba2
  - 指南卡片：#e8eaf6 → #e1f5fe

### 背景色
- **主背景**: #ffffff
- **次级背景**: #fafafa
- **悬停背景**: #e6f7ff

### 文字色
- **主文字**: rgba(0, 0, 0, 0.88)
- **次级文字**: #8c8c8c
- **禁用文字**: #d9d9d9

### 边框色
- **默认**: #e8e8e8
- **浅色**: #f0f0f0
- **深色**: #d9d9d9

## 🔄 交互动效

### 1. 悬停效果
- **卡片**: `transform: translateY(-2px)` + 阴影
- **指南项**: `transform: translateX(-2px)` + 阴影
- **侧边栏项**: 背景色变化 #e6f7ff

### 2. 过渡动画
- **通用**: `transition: all 0.3s`
- **快速**: `transition: all 0.2s`

### 3. 聚焦状态
- 输入框边框色变化
- 添加蓝色阴影：`box-shadow: 0 2px 8px rgba(24, 144, 255, 0.1)`

## 📱 响应式设计

### 大屏（>1024px）
- 显示完整三栏布局

### 中屏（768px-1024px）
- 可考虑隐藏右侧指南面板
- 保留左侧边栏和主内容

### 小屏（<768px）
- 左侧边栏改为抽屉模式
- 全屏显示主内容
- 隐藏右侧指南面板

## 🎯 设计原则

1. **简洁优先**: 去除不必要的边框和背景
2. **层次分明**: 通过间距和颜色区分内容层级
3. **交互友好**: 悬停反馈、聚焦状态清晰
4. **信息密度**: 合理的留白和内容密度
5. **一致性**: 统一的圆角、间距、颜色使用

## 🔧 关键样式参数

```css
/* 圆角 */
--border-radius-small: 4px;
--border-radius-medium: 8px;
--border-radius-large: 12px;

/* 间距 */
--spacing-xs: 4px;
--spacing-sm: 8px;
--spacing-md: 12px;
--spacing-lg: 16px;
--spacing-xl: 24px;

/* 阴影 */
--shadow-small: 0 2px 8px rgba(0, 0, 0, 0.08);
--shadow-medium: 0 4px 12px rgba(0, 0, 0, 0.1);
--shadow-focus: 0 2px 8px rgba(24, 144, 255, 0.1);

/* 过渡 */
--transition-fast: all 0.2s;
--transition-normal: all 0.3s;
```
